1. 安装 Fish Shell#

步骤#

  1. 更新系统：

    1

    sudo apt update && sudo apt upgrade -y

  2. 安装 Fish：

    1

    sudo apt install fish -y

  3. 验证安装：

    1

    fish --version

2. 设置 Fish 为默认 Shell#

  1. 确认 Fish 的路径：

    1

    which fish

    通常路径为 /usr/bin/fish。

  2. 将 Fish 设置为默认 Shell：

    1

    chsh -s /usr/bin/fish

  3. 重新启动终端以应用更改。

3. 安装 Fisher (Fish 的插件管理器)#

Fisher 是一个流行的插件管理器，用于轻松安装和管理 Fish 插件。

  1. 安装 Fisher：

    1

    curl -sL https://git.io/fisher | source && fisher install jorgebucaran/fisher

  2. 验证安装：

    1

    fisher --version

4. 安装常用插件#

推荐以下常用插件：

  1. 插件安装命令：

    1

    fisher install <插件名>

  2. 推荐插件列表：

    • oh-my-fish/theme-bobthefish: 美观的主题。

      1

      fisher install oh-my-fish/theme-bobthefish

    • jorgebucaran/nvm.fish: Node.js 版本管理。

      1

      fisher install jorgebucaran/nvm.fish

    • PatrickF1/fzf.fish: 模糊查找。

      1

      fisher install PatrickF1/fzf.fish

    • jethrokuan/z: 快速目录跳转。

      1

      fisher install jethrokuan/z

    • franciscolourenco/done: 长时间命令完成时提醒。

      1

      fisher install franciscolourenco/done

5. 配置 Fish Shell#

Fish 使用 $HOME/.config/fish/config.fish 文件来存储配置。可以编辑该文件以自定义你的 Shell。

  1. 打开配置文件：

    1

    nano ~/.config/fish/config.fish

6. 美化 Shell#

  1. 安装 Nerd Fonts: Nerd Fonts 提供丰富的图标支持。可以从 Nerd Fonts 官网 下载适合你的字体，并安装到系统中。

  2. 更改终端字体：

    • 打开终端设置。
    • 在外观设置中选择刚刚安装的 Nerd Font 字体。

  3. 启用主题：

    • 使用 bobthefish 或者 tide 主题进行美化：

      1

      fisher install ilancosman/tide

    • 运行 tide configure，根据提示完成配置。

7. 启用自动补全和语法高亮#

  1. Fish 自带强大的自动补全功能，可以直接使用。
  2. 如果需要更高级的语法高亮，可以使用插件 tide。

8. 安装常用工具#

  1. fzf (模糊查找工具):

    1

    sudo apt install fzf -y

  2. ripgrep (快速搜索工具):

    1

    sudo apt install ripgrep -y

9. 备份和恢复配置#

备份#

将 ~/.config/fish 目录打包保存：

1

tar -czvf fish-config-backup.tar.gz ~/.config/fish

恢复#

将备份的文件解压到对应目录：

1

tar -xzvf fish-config-backup.tar.gz -C ~/

10. 测试配置#

  1. 打开一个新的终端，确认配置生效。

  2. 运行以下命令，验证所有功能：

    1
2
3

    fish
fisher list
echo $PATH
